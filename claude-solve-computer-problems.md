## 真实案例：Mac Mini 装 Windows 11 找驱动 📖

**背景**：2011 年的 Mac Mini 装 Windows 11 玩 Steam。结果——**网卡识别不了！** 没网就是砖头 💔

**传统方式要折腾多久？**
1. 百度搜索 → 10 个帖子说法不一
2. 试 Apple 官网 → 404 链接失效
3. 下第三方驱动 → 700MB 下完发现不对
4. 再试第二个、第三个...
5. **折腾 2-3 小时，还不一定成功** ⏰😤

---

**我只说了一句话**：

- 📝 生成了 3 份完整文档

**我找到了驱动后**，Claude 立刻：
- 📝 记录到文档
- 📚 生成完整安装指南
- 🔗 整理所有备用方案
- ✅ 形成可分享的完整文档

## 怎么和 Claude 说话效果最好？ 💬

### 1. 说清楚 4 个点

❌ **模糊**：电脑坏了，帮我看看
✅ **清晰**：Mac Mini 2011 装 Windows 11，网卡识别不了

**记住 4 点**：
1. **设备/系统**：什么设备？什么系统？
2. **问题现象**：出了啥问题？
3. **试过没**：自己试过啥？
4. **想干啥**：最终要达到啥效果？

---

### 2. 利用 shell 目录

根据 `shell/CLAUDE.md` 规则：
- ✅ Claude 可以在 shell 目录**随意读写**
- ✅ 适合存临时文件、实验代码

**最佳实践**：
```
把驱动下载到 ~/Sites/shell/ 目录
并生成安装说明文档
```

---

### 3. 要多个备选方案

❌ **只要一个**：给我一个解决方案
✅ **要备选**：请提供 2-3 个方案，包括备选

**好处**：主方案失败还有备选

---

### 4. 让 Claude 生成文档

❌ **只解决问题**：找到驱动就行
✅ **同时生成文档**：顺便生成完整文档，包括安装步骤、常见问题

**好处**：
- 日后查阅
- 分享给别人
- 不用再次搜索

## 实用命令模板 📋

### 找驱动
```
帮我找 [设备型号] 在 [系统] 上的驱动
要求：官方优先，提供备用地址，给出安装步骤
```

### 写脚本
```
帮我写个脚本，实现：
1. [功能]
2. [功能]
要求：包含错误处理，加详细注释
```

### 配置系统
```
帮我配置 [系统/软件]，需要：
1. [配置项]
2. [配置项]
要求：用国内镜像（如适用），自动化脚本
```

### 诊断问题
```
我的 [系统/软件] 出现问题：
[错误信息]

已尝试：
1. [尝试的方案]

请帮我诊断并提供解决方案
```

## 总结 🎯

用 Claude Code 解决电脑问题的核心：

1. **说清楚** → 设备、系统、问题、目标
2. **要完整** → 不只要答案，还要文档和备选
3. **用 shell** → 临时文件放这里，AI 随意操作

**最重要的**：把 Claude 当作**技术助理**，不是魔法师。它帮你搜索、整理、生成文档，但最终操作还是你来做！
